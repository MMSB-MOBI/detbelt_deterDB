How To Use The Detergent Database CRUD Interface

Last modified: 17 dec. 2017

The menu bar

Title button : Detergent Database CRUD Interface

The first button from the left allow reload the home page.

Load database

The Load database button only displays the table containing detergents. Click again will reload the table from database.

The table displays 4 columns by default: _id, volume, color, and category. The other available columns can be displayed with the Hide/Show columns button.

Manage detergent

There are a few buttons that allow you to modify the content of the the database:
  • Insert new detergent

    • This button adds a new detergent to the database. When clicked, an insert form appears, with fields to fill. It is important to note that:

    • _id fields correspond to the name of the detergent.
    • The values of _id, volume, color and category fields are mandatory: insertion will not be possible without them!
    • The value of volume must contain one . and not a , if it has decimal values

    The Add new attribute button adds two input fields corresponding to the name of new attribute and his value for current detergent. To submit a new attribute, you must fill these two fields. (It's is currently not available because it's in deloppement)

  • Update detergent

    • This pannel allows you change one or many value for every single detergent. To proceed, choose one detergent on the table, then the fields of it's attributes will appear. Note that value of _id can't be modified because it identifies the unique detergent.

  • Remove detergent

    • To remove a detergent simply select the detergent in question, then click on the button Confirm. The manipulation being irreversible, a message appears and asks for confirmation.

Table of data

The data table is available from the Load database tab. It displays all the detergents stored in the database at the moment the user looks at it. That is, if a new detergent is added to the base then it will be displayed; similarly, a detergent that has been removed will no longer be present.

Sorting

The database is initially sorted in alphabetical order on the names of detergents (_id). It is possible to sort on another column by clicking on its name. By default, the table displays the first 10 detergents ('Show 10 entries') and it is possible to modify this field to display up to 100 detergents per page. The others are in the following pages accessed by the numbers at the bottom right of the table.

Search

It is possible to search for an element in the table with the search bar at the top right of the 'Search' table.

Hide/Show columns

Through this button you can toggle columns display panel. Check the names of columns you want see and press Apply to validate.
