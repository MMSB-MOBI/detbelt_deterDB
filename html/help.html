Add new detergent

The Add new detergent button adds a new detergent to the database. When clicked, an insert form appears, with fields to fill. It is important to note that:

  • '_id' fields correspond to the name of the detergent
  • '_id', 'volume', 'color' and 'category' fields are mandatory: insertion will not be possible without
  • The volume of the field 'must contain one'. 'and not a'; ' if it's not an integer

The Add new attribute button adds a new field whose name is not in the name (New attribute name) and value (Value for new attribute).

The new detergent is submitted with the Add new detergent button.
