How To Use The Detergent Database CRUD Interface

Last modified: 11 dec. 2017

The menu bar

Title button : Detergent Database CRUD Interface

The first button from the left allow reload the home page.

Load database

The Load database button only displays the table containing detergents. Click again will reload the table from database.

The table displays 4 columns: _id, volume, color, and category. The other available columns can be displayed with the Select which columns to display button.

Manage detergent

There is few buttons allow you modify the content of database:
  • Insert new detergent

    • This button adds a new detergent to the database. When clicked, an insert form appears, with fields to fill. It is important to note that:

    • _id fields correspond to the name of the detergent.
    • The values of _id, volume, color and category fields are mandatory: insertion will not be possible without them!
    • The value of volume must contain one . and not a , if it has decimal values
    The Add new attribute button adds two input fields corresponding to the name of new attribute and his value for current detergent. To submit a new attribute, you must fill these two fields.

  • Update detergent

    • This pannel allows you change one or many value for every single detergent. To proceede, choose one detergent on the table, then the fields of it's attributes will appear. Note that value of _id can't be modified because it identifies the unique detergent.

  • Remove detergent

    • To remove a detergent simply select the detergent in question, then click on the button Confirm. The manipulation being irreversible, a message appears and asks for confirmation.
